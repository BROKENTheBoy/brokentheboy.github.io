用上这些软件，Markdown编辑效率增加100%！

发表于 2022-01-12 更新于 2022-01-13

大家好，我是BROKENTheBoy！我的博客开通啦！今天这篇文章，我会给大家推荐几个我用过的，觉得不错的Markdown编辑器。废话少说，让我们开始吧！

1.Typora

一说起Markdown，许多人都会想到Typora。Typora作为一款Markdown编辑器，不会像有道云笔记那样功能繁琐，Typora界面十分简洁，十分适合专门写Markdown文章的程序员使用。

2.Mark Text

Typora也开始收费了，以下有三种选择给你：1.购买，2.使用旧版，3.使用新的Markdown编辑器。如果你选择第三种，那么这款Mark Text绝对符合你的要求。Mark Text在功能上，界面上几乎就是第二个Typora，而且Mark Text完全免费，是一个开源项目。

3.Visual Studio Code

这个软件几乎是所有学编程的都认识。Visual Studio Code是一款由微软开发的编程软件，可以安装许多插件。可用它来写Markdown文章并不是实时预览，你还要去安装相关的插件。在这里我推荐两个插件：Markdown All in One和Markdown Preview Github Styling。

4.总结

这篇文章我把我用过的Markdown编辑器全部推荐了一遍，如果还有时间的话，我也会继续找出一些比较优质的Markdown编辑器分享给大家。好了，今天这篇文章就到这里。拜拜！
